# ethereum-example
Example for ethereum and web3 js

#Running:
```bash
npm install --save-prod express

docker-compose up -d 
```

#Test from url:
http://0.0.0.0:3000/

#Stop
```bash
Control + C
```

#Check logs:
```bash
docker-compose logs -f
```

#Exec to app in docker
```bash
docker exec -it app bash
```
